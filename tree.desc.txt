根据 tree.desc.txt 文件了解项目大致结构

windows 系统

你需要先查看和了解
README.md
tree.txt
tree.dir.txt
这三个文件包含了整个项目的目录结构

tree.dir.txt 文件:
 由 dir /o /b > tree.dir.txt 命令生成 只包含了基本的目录和文件结构,不包含子目录

tree.txt 文件
    由 tree-cli treee -l 10 -o tree.txt --ignore ".git,.github,.husky,.next,.vscode,node_modules,dist,*.log,*.tmp" 命令生成
    包含了整体的目录结构,也包含子目录和子文件

项目架构

shadcn+next.js 

This project using the following stack:

Framework - Next.js 15
Language - TypeScript
Styling - Tailwind CSS v4
Components - Shadcn-ui
Schema Validations - Zod
State Management - Zustand
Search params state manager - Nuqs
Auth - Clerk
Tables - Tanstack Data Tables • Dice UI
Forms - React Hook Form
Command+k interface - kbar
Linting - ESLint
Pre-commit Hooks - Husky
Formatting - Prettier
